Summary: Lattitude vs. X

The purpose of these plots which come from a previously conducted weather analysis using data from a plethora of locations. We studied at that time what was the current weather conditions all over the world. We randomly chose a minimum of 500 cities using these cities locations via their latitude and longitude coordinates. We used API calls to acquire this weather data from https://openweathermap.org/api, from this point we analyzed and manipulated the data, and made the weather data into DataFrames, which are essentially tables with rows and columns with given column headers like Maximum Temperature, Humidity, Cloudiness, and Wind Speed. We then used the Data at our hands to nmake these plots you see in front of you. Patterns may be observed as we analyze these cities' latiudinal and logitudinal coordinates to draw some conclusions about the dependence of weather in relation to these coordinates and hence the proximity to the equator, etc.

Here we presented the results in a web visualization dashboard. This is a user-friendly way to access the results we reached in our analysis. For further technique detailes, please inquire regarding more detailed information regarding our weather analysis project where we used Python (a computer program) and Jupyter Notebook to serve as our program to code in, Jupyter Notebook is an open-source web application that allows you to create and share documents that contain live code, equations, visualizations and narrative text. Uses include: data cleaning and transformation, numerical simulation, statistical modeling, data visualization, machine learning, and much more.
